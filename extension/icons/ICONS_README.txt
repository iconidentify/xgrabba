XGrabba Extension Icons

Required icon files:
- icon-16.png (16x16 pixels)
- icon-48.png (48x48 pixels)
- icon-128.png (128x128 pixels)

Design guidelines:
- Use a simple download/archive icon concept
- Colors should work on both light and dark backgrounds
- Primary color: #1D9BF0 (X.com blue)

To generate icons, you can:
1. Create an SVG design and export to PNG at different sizes
2. Use a tool like Inkscape, Figma, or online icon generators
3. The icon should represent video archiving/downloading

Suggested icon concept:
- A downward arrow entering a box/folder shape
- Simple, recognizable at small sizes
- Consistent with the button icons used in the extension
